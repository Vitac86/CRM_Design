CRM static-uikit review launcher

Локальная стартовая страница для просмотра standalone demo pages и UMI extraction packs.

Standalone demo pages

  • Dashboard — операционный обзор.
  • Subjects — реестр субъектов.
  • Subject card — карточка субъекта.
  • Subject register — регистрационная форма субъекта.
  • Contract wizard — мастер оформления договора.
  • Brokerage — операции брокерского направления.
  • Trust management — управление доверительными активами.
  • Agents — список и статусы агентов.
  • Requests — поручения.
  • Compliance — очередь комплаенса.
  • Compliance card — карточка проверки комплаенса.
  • Middle-office clients — журнал клиентов.
  • Middle-office reports — журнал отправленных отчётов.
  • Depository — депозитарные операции.
  • Back office — бэк-офисные задачи.
  • Trading — торговый обзор.
  • Trading card — карточка торгового кейса.
  • Administration — администрирование системы.
  • Archive — архивные записи.
  • Error — шаблон страницы ошибки.

npm run build
python -m http.server 8080

http://localhost:8080/static-uikit/index.html

Safety constraints

  • No React inside static-uikit
  • No Vite build for static-uikit
  • No CDN / Google Fonts / external API
  • No telemetry/metrics trackers
  • No fetch/XHR/WebSocket/localStorage/sessionStorage
  • Standalone pages remain in static-uikit/pages
  • UMI packs are extraction packs, not runtime
