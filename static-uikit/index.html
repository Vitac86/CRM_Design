CRM Static UI Kit

Review launcher

Локальная стартовая панель для standalone demo pages, UMI extraction packs и handoff-валидации без React/Vite runtime.

local-only no CDN no API calls
Запуск из корня репозитория: cd CRM_Design && python -m http.server 8080 → http://localhost:8080/static-uikit/index.html
Запуск из папки static-uikit: cd CRM_Design/static-uikit && python -m http.server 8080 → http://localhost:8080/index.html

Local validation

manual run

Команды запускаются вручную из терминала (это текстовая памятка, не browser automation).

node static-uikit/tools/validate-static-uikit.mjs
npm run build
python -m http.server 8080

URL при запуске сервера из корня: http://localhost:8080/static-uikit/index.html
